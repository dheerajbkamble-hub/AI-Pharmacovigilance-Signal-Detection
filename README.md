# AI-Pharmacovigilance-Signal-Detection
AI-based pharmacovigilance project integrating LLM-driven clinical reasoning with statistical signal detection (PRR/ROR) and data visualization for adverse drug reaction analysis.

## 🧪 Methods

### Statistical Analysis
- Proportional Reporting Ratio (PRR)
- Reporting Odds Ratio (ROR)

These methods are widely used in pharmacovigilance signal detection.

### AI-Based Simulation
- LLM-based clinical reasoning
- Severity classification
- ADR interpretation

### Data Source
- FAERS-inspired dataset
- Drug-event pair analysis

- ## 🧠 Interpretation

- Statistical methods identify signal strength
- AI provides contextual clinical reasoning
- Hybrid approach improves decision-making

This reflects emerging trends in AI-driven pharmacovigilance systems.
